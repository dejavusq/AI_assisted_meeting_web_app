# 腾讯云语音识别配置指南

本指南帮助你完成腾讯云语音识别（ASR）服务的配置，以便在 déjà vu 会议妙记中使用实时语音转录功能。

---

## 前置条件

- 一个腾讯云账号（[注册地址](https://cloud.tencent.com/register)）
- 已完成实名认证

---

## 第一步：开通语音识别服务

1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/)
2. 在顶部搜索栏搜索 **「语音识别」**，或直接访问 [语音识别控制台](https://console.cloud.tencent.com/cas/asr)
3. 点击 **「立即开通」**（首次使用需要同意服务协议）
4. 语音识别提供 **免费额度**，具体以腾讯云官网当前政策为准

---

## 第二步：获取 App ID

1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/)
2. 点击右上角头像 → **「账号信息」**
3. 在账号信息页面找到 **「AppID」**，复制该号码

> AppID 是一串纯数字，共 10 位。

---

## 第三步：获取 Secret ID 和 Secret Key

1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/)
2. 点击右上角头像 → **「访问管理」**
3. 在左侧菜单选择 **「访问密钥」** → **「API 密钥管理」**
4. 点击 **「新建密钥」**（建议只创建一个，妥善保管）
5. 创建后会显示：
   - **SecretId**（形如 `AKID` 开头的一串字母数字）
   - **SecretKey**（一串字母数字）

> ⚠️ **SecretKey 只在创建时显示一次**，请立即复制保存。如果遗失，需要删除旧密钥并重新创建。

---

## 第四步：在应用中配置

1. 打开 déjà vu 会议妙记
2. 点击左侧导航栏底部的 **⚙️ 设置** 图标
3. 找到 **「腾讯云语音识别」** 配置卡片
4. 填写以下三个字段：

   | 字段 | 说明 |
   |------|------|
   | App ID | 账号信息中的 AppID（10位纯数字） |
   | Secret ID | API 密钥管理中的 SecretId |
   | Secret Key | API 密钥管理中的 SecretKey |

5. 点击 **「保存」**
6. 状态显示成功表示配置完成

---

## 第五步：验证配置

1. 回到首页，点击 **「开始录音」**
2. 对着麦克风说几句话
3. 如果转录区域实时显示文字，说明配置成功

---

## 常见问题

### Q：配置后提示连接失败？

- 检查 App ID、Secret ID、Secret Key 是否填写正确
- 确认 Secret Key 没有多余的空格
- 确认腾讯云账号已开通语音识别服务

### Q：录音开始后没有文字显示？

- 检查浏览器是否允许了麦克风权限
- 确认使用的是支持 MediaDevices API 的现代浏览器（Chrome / Edge / Firefox）
- 尝试刷新页面后重新录音

### Q：数据安全吗？

- 所有配置信息仅保存在当前浏览器的本地存储中
- **不会**上传到任何第三方服务器
- 在公共设备上使用时，请勿勾选「保存密钥」，用完后及时清除浏览器数据

### Q：SecretKey 遗失了怎么办？

1. 登录腾讯云控制台 → 访问管理 → API 密钥管理
2. 删除旧密钥
3. 新建密钥，重新获取 SecretId 和 SecretKey
4. 在应用设置中更新配置
